How to generate random number in c++

rand()

How to use rand()

rand() generates random number.Its range is 0 to 32767.It may depend on the computer so to get the highest random number we can use macro RAND_MAX which is available in cstdlib 

    
#include<iostream>
#include<cstdlib>

int main(){
int random;
for(int i = 0;i<=10;i++){
random = rand();
cout<<random<<endl;
}
cout<<"the random max is"<<RAND_MAX<<endl;

return 0;
}
In the first line an integer named random is defined
Running the loop for 10 times
rand() generates 10 random numbers and store it in integer random It is always constant for specific computer.In this case if we run this program 1000 times also it will return same number for that time.So it is random for first time
on the last line RAND_MAX is called which returns the highest random number possible. It is available in cstdlib

How to generate random number differently in every time in c++

using srand we can generate random number differently 

    
#include<iostream>
#include<ctime>

int main(){
int random;
srand(time(0));
for(int i = 0;i<=10;i++){
random = rand();
cout<<random<<endl;
}


return 0;
}
srand(time(0)) helps us to generate random number in every second
rand() generates 10 random numbers and store it in integer random. In this case if we run this program 1000 times it will return different number every time.So it is random

How to generate random number differently in every time between range in c++

    
#include<iostream>
#include<ctime>

int main(){
int random,high = 10,low = 1;
srand(time(0));
for(int i = 0;i<=10;i++){
random = rand()%high+low;
cout<<random<<endl;
}


return 0;
}
srand(time(0)) helps us to generate random number in every second
rand()%high means that random number is generated and produces the remainder when rand() is divided by high
after adding low
rand()%high is added to produce random number between low and high In this case if we run this program 1000 times it will return different number every time between 1 and 10 as value of high is 10 and low is 1. So it is also truly random.
