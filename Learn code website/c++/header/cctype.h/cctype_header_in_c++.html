What are the functions available in cctype.h in c++

  1. isdigit
  2. isxdigit
  3. isalpha
  4. isalnum
  5. iscntrl
  6. isgraph
  7. tolower
  8. toupper

isdigit

bool

How to use isdigit

isdigit is the function provided in cctype.h header file in c++ As the name suggests it is used to find whether the given character is digit or not. It will return true if it is digit else will return false. Characters like '1' , '8' , '20' will return true 

            
char allData[11] = {'a','B',' ','#','\t','5','m','L','\n','0'};
for(int z = 0;z<11;z++){
    if(isdigit(allData[z])){
        cout << allData[z]<< " is digit"<<endl;
    }
}
In the first line array of character is defined.
Running the loop for all characters in the array
If allData[z] is a digit then it will return true and and the code inside if will be executed 


5 is digit
0 is digit

tolower

int

How to use tolower

It converts any letter to lowercase letters. It returns number of ASCII value. Insert any character between tolower and you will get the answer. 

            
char allData[6] = {'a','B','C','c','t'};
char ans;
for(int z = 0;z<6;z++){
if(isalpha(allData[z])){
ans = tolower(allData[z]);
cout << ans<<endl;    
}}
In the first and second line array is defined,first one is of characters and second one is for storing answer.
Running the loop for all characters in the array
If allData[z] is a alphabet then it will return true and and the code inside if will be executed
tolower returns integer value i.e. ASCII value. Since it is stored in characters it will be converted into character so character is displayed.
For Eg:
tolower('A') returns 97 which becomes a when stored in char
